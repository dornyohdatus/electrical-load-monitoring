#Electrical Load Monitoring and Billing Simulator

## Course
EEE 227 - Introduction to C++ Programming

## Programme
HND Electrical Engineering Level 200

## Project Description
This project simulates electrical load monitoring and electricity billing for a household or small facility.

The system allows users to:
- Register electrical appliances
- Store appliance power ratings
- Track usage hours
- Calculate energy consumption
- Estimate electricity cost
